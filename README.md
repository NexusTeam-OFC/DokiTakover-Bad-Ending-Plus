## ✦ About

Bad Ending Plus is a story-driven FNF mod focused on:

- Psychological horror
- Disturbing atmosphere
- Emotional storytelling
- Custom visuals
- Original soundtrack
- Meta elements

## ✦ Installation

1. Download the mod

2. Extract into:

```txt
PsychEngine
│
├── mods
│      └── Bad Ending Plus
```

3. Launch Psych Engine

4. Enable the mod inside the Mods Menu

Option 2:
1. Download the PsychEngine that the mod already provides.

2. Extract Into Some Folder

3. Launch PsychEngine
  
4. And Have Fun !
